# WeedDetection
## 北京理工大学2024年秋计算机视觉  
从 https://pan.best8.eu.org:10443/s/Npsn 下载杂草数据集（田间杂草（4种）和农作物（1种）），对数据集进行预处理后（降噪，数据增强后），按8：2划分训练与测试集，基于yolov11模型进行训练与测试。最终模型在测试集上的准确率可达60%+（一般小组为50%），班级小组排名约为5/50,效果良好；
